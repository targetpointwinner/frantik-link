# frantik.by — кастомный link-in-bio

Замена Taplink: премиум-посадка магазина детской и подростковой одежды.
Один самодостаточный `index.html` (без сборки, без фреймворков) + каталог школьной коллекции, который тянется с Яндекс.Диска.

## Что внутри
- **Хиро:** смайл-медальон, фирменный леттеринг `FRANTIK.BY`, кнопка на каталог.
- **Магазины:** 3 точки (Тивали, Титан, Экспобел). У каждой — «Как добраться» (карта) и «Позвонить».
- **Каталог «Школьная коллекция 2026/2027»:** табы Девочки / Мальчики, сетка с ленивой подгрузкой («Показать ещё»), лайтбокс со свайпом.

## Структура
```
index.html          — вся посадка (дизайн + логика)
catalog.json        — манифест каталога (генерируется скриптом)
catalog/girls/*.webp, catalog/boys/*.webp — фото (генерируются скриптом)
sync.py             — синхронизация фото с Яндекс.Диска
assets/logo.png, mark.png, wordmark.png    — логотип (вырезан из фирменного файла)
assets/stores/*.mp4 — видео витрин магазинов (добавить вручную, см. ниже)
```

## Видео магазинов
Положи ролики витрин с такими именами (mp4, вертикальные, лучше без звука, до ~10 сек):
```
assets/stores/tivali.mp4
assets/stores/titan.mp4
assets/stores/expobel.mp4
```
Они сами зациклятся в шапке карточки. Пока файла нет — показывается градиент с названием точки.

## Обновить каталог с Яндекс.Диска
Папка-источник задана в `sync.py` (`PUBLIC_KEY`). Сейчас:
`https://disk.yandex.ru/d/u7Tb05aUevhcqA` → `Для девочек` / `Для мальчиков`.

```bash
python3 sync.py            # докачать новое (быстро, пропускает уже скачанное)
python3 sync.py --force    # перекачать всё заново
```
Скрипт берёт JPEG-превью Яндекса (оригиналы в .heic браузеры не показывают), жмёт в WebP ~1000px и пишет `catalog.json`. Требуется `pillow`: `pip3 install pillow`.

## Поменять адреса/телефоны магазинов
В `index.html` найди массив `STORES` — там имя, адрес, часы, ссылка на карту, телефон, контактное лицо.

## Локальный предпросмотр
```bash
python3 -m http.server 8791
# открыть http://localhost:8791
```

## Деплой (Netlify)
Проще всего: зайти на app.netlify.com → «Add new site» → «Deploy manually» → перетащить папку `frantik-link`.
Дальше в настройках домена подключить свой адрес и поставить ссылку в Instagram-био вместо Taplink.
